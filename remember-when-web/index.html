Remember When

Create a group and start sharing; your agent will create a log of what happens. Use it for work, travels, or friend groups... It's easy, fast, and free.

What is this AI Skill?

It is a professional toolkit designed for AI Agents (OpenClaw). It enables your agent to act as a proactive digital archivist within WhatsApp or Telegram groups, identifying valuable moments and persisting them to local storage using our dedicated CLI.

Autonomous Media Handling

The agent detects photos, videos, and voice notes, copies them to date-organized folders (YYYY-MM-DD), and ensures your data stays private on your machine.

Cognitive Summaries

Your AI agent generates enriched chronicles and daily summaries, allowing you to recall complex conversations through semantic meaning rather than just keywords.

Installation

1. Install the storage engine (CLI) globally:

npm install -g remember-when-cli

2. Configure the Skill in your environment:

Choose your preferred installation method:

Install directly using the Clawhub CLI:

npx clawhub install remember-when-skill

Alternatively, use the automated installer script:

npx skills add https://github.com/2mes4/remember-when --skill remember-when-skill

Install via the standard OpenClaw command:

openclaw skills install remember-when-skill

3. Configure your OpenClaw channels:

Activate the groups you want the agent to monitor and set requireMention: false. This grants the agent permission to observe all group activity without needing to be @mentioned.

4. Let your Agent handle the rest:

Your agent will automatically start archiving memories. Use the inventory command to audit your local memory state:

remember-when inventory
