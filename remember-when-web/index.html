Remember When

A smart system to capture and organize your digital memories directly on your local disk.

What is Remember When?

It is a hybrid solution composed of an OpenClaw Agent and a Node.js CLI. The agent acts as your personal archivist in WhatsApp or Telegram groups, detecting memorable moments and automatically saving them to your computer.

Multimedia Management

Photos, videos, and voice notes are automatically copied to folders organized by date (YYYY-MM-DD) in your home directory.

Smart Summaries

The agent doesn't just save files; it generates enriched summaries and daily chronicles so you can search for moments by their meaning, not just by date.

Installation

1. Install the storage engine (CLI) globally:

cd remember-when-cli
npm install -g .

2. Configure your Skill in OpenClaw:

You can automatically add this skill to your agent using skills.sh with the following command:

npx skills add https://github.com/2mes4/remember-when --skill remember-when-skill

(Replace with your repository's final URL if necessary).

3. Let the magic happen:

The agent will start saving entries. You can check the inventory of what's missing with the command:

remember-when inventory
